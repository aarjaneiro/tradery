The Info class

A very important class that constitutes a base class for all plug-ins, plug-in configurations and others is the Info class. 

This stores the basic information about any object that is to be recognized by the platform: Name, Description and UniqueId. 

Name and Description are regular strings that can contain any characters, and they provide descriptive information about the object that is normally presented to the user.

 UniqueId which is a UUID value and is unique for each plug-in, configuration, runnable etc. is used to identify each component within the system.

class Info
{
public:
    // constructor
  Info( const UniqueId& id, const t_string& name, const t_string& description );
    // copy constructor
  Info( const Info& info );
    // default constructor - generates a new unique id
  Info();

  const UniqueId& id() const;
  const t_string& name() const;
  const t_string& description() const;
 }

Note: when developing a new plug-in or configuration, a new UniqueId can be generated using the UUID tool provided with Visual Studio .net, or the default constructor of the class UniqueId.
