Slippage derived classes are used in back-testing to simulate the effect of slippage on trading systems. 

Besides the plug-in API, a slippage plug-in must implement a Slippage interface.

The method getValue is called by the platform and it must return the slippage calculated based on the parameters passed to it such as shares, volume and price. Other relevant parameters may be added in the future. The returned value is an absolute value, of whatever type is relevant to the system being run, such as dollars, or other currencies.

class Slippage : public Info
{
  virtual double getValue( unsigned long shares, unsigned long volume, double price ) const = 0;
};
