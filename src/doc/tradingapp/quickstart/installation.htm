Run setup.exe to install the TradingApp Platform on your computer. The package consists of:

  • the binary components for the Integrated Visual Trading Environment, platform components, various plug-ins and the standalone Yahoo historical quotes downloader, in the bin\releasep directory

  • the documentation files including the API help, developer guides, license agreement etc in the doc directory 

  • the library (*.lib) files required for building various plug-ins in lib\releasep. 

  • several Visual Studio .net 2003 projects for the various sample plug-ins.

    • dyndsplugin a sample dynamic data source plugin

    • fileplugins - file based plugins: data source, symbols source and signal handler

    • systemsplugin1 - several sample trading systems, slippage and commission plugins

    • systemsplugin2 - more sample trading systems

  • sample data and symbols for the 3 data file formats currently supported, in the data directory 

  • an empty directory "signal files" where the file signal handler will save signal files by default

  • several sample session files in the sessions directory.

  • auto updater

  • uninstaller

 

Note: many new features are under development, and this document as well as the application are expected to evolve and change significantly over time, so check back for new releases.

Make sure to run the auto-updater periodically, either from the Help/Check for updates menu or from the Applications menu
