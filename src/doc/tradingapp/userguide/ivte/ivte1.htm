Main Window (Workspace)

The workspace consists of:

  • Menu and toolbar - offers access to various commands

  • Plug-in Explorer bar - shows all the available plug-ins, in a tree form

  • Properties bar – shows properties of selected object

  • One or more session windows

Note: the workspace can be re-organized at will by moving any of the toolbars in various positions and configurations, docked, undocked etc. 2 or more of them can be combined to form tabbed toolbars, or can be set to enter auto-hide mode by clicking on the pin button on the title bar once they are in docked position. The same is true for session windows. The workspace status (bars positions, sizes etc) is saved between runs.

Plug-in Explorer Bar

The plug-in explorer is populated during application start-up and shows all available plug-in types, plug-in instances and their configurations.
