The File Data Source plug-in (FD) serves two purposes: it is a functional plug-in used for retrieving data stored in files, and it is also a programming sample illustrating various techniques for how to write a plug-in in general and a data source plug-in in particular.

The FD is described it from a functional and user interface standpoint. For more details on the programming aspects, please refer to various Dev Guides.

The FD is normally listed in the Plug-in Explorer under DataSource Plug-ins/File Data Source. A FD configuration will retrieve the the data for a set of symbols.

The FD makes certain assumptions about the data files:

  • each symbol corresponds to a file with the name being the symbol
  • all files are located in the same directory
  • all files have the same extension
  • all files have the same format
