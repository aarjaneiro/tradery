The File Symbols Source plug-in (FSSP) serves two purposes: it is a functional plug-in used to connect to list of symbols stored in files, and it is also a programming sample illustrating various techniques for how to write a plug-in in general and a symbols source plug-in in particular.

Here we'll describe it from a functional and user interface standpoint. For more details on the programming aspects, please refer to various Dev Guides.

The FSS is normally listed in the Plug-in Explorer under SymbolsSource Plugins/File Symbols Source. A FSSP configuration will serve a list symbols stored in a file.
