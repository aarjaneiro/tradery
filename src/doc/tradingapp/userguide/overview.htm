The TradingApp 2.0 platform consists of the following main components:

  1. The trading framework which implements most of the trading and trading helper services used by all other components, exposed as C++ APIs.

  2. The plug-in framework, which implements the services used to extend the platform's functionality with external components, exposed as C++ APIs.

  3. The Integrated Visual Trading Environment (IVTE) which integrates all the platform elements together into coherent trading setups, in an user friendly manner

 

The first two will be described at a conceptual level here, and more in detail in the various Dev Guides and the API help.
